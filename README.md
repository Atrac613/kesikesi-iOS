kesikesi for iPhone
=============

kesikesi(けしけし)はカメラで撮影した写真に自由にマスクをかける事ができる iPhone アプリです。

写真は Twitter に投稿ができ、マスクはこのアプリの利用者のみ解除ができます。

また、マスクを解除するモードは、スクラッチ、傾き、端末を振る、音の大きさが用意されています。
